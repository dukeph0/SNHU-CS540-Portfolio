# SNHU CS 540: A Portfolio of Work

Welcome! This repository contains my collected works from the CS 540: Data Literacy & Visualization course at Southern New Hampshire University. It serves as a showcase of the skills, knowledge, and practical applications I've developed throughout my studies.

## About This Portfolio

The purpose of this portfolio is to demonstrate my journey through the course curriculum. Here, you will find a curated selection of my projects, assignments, and other coursework. Each piece has been chosen to highlight my understanding of key concepts and my ability to apply them to solve complex problems.

## Navigating This Repository

This repository is organized to be clear and easy to follow. You can explore the different projects and assignments by navigating through the folders. Each project folder contains the source code, along with any relevant documentation or notes that explain the work in more detail.

I invite you to explore my work and see the progress I've made. Thank you for visiting!
